Acessando o Protocolo

Abra o link do Protocolo de Testes que você recebeu — pelo WhatsApp ou e-mail. Ele funciona em qualquer navegador (Chrome, Safari, Edge), no computador ou no celular.
1
Clique no link recebido. A página abre direto — não precisa fazer login, o formulário é independente da plataforma Juris Code.
2
À esquerda fica o menu de navegação: os 8 módulos de teste e a barra de progresso geral.
3
No topo ficam as instruções gerais e o resumo com os contadores de Aprovados / Reprovados / Pulados / Pendentes.

Abrindo um Caso de Teste

Cada teste tem um código (ex: TC-001), um título e uma etiqueta de prioridade. Clique em qualquer parte da linha para expandir e ver os detalhes.
Antes de clicar — fechado
TC-006 Pergunta jurídica básica (civil) Alta ▾
↓ clique no título ↓
Depois de clicar — aberto
TC-006 Pergunta jurídica básica (civil) Alta ▾
Pergunta a digitar
"Qual o prazo prescricional para ação de cobrança de honorários advocatícios?"
✓ Critérios de Aprovação
✓ Cita o Art. 206, §5º, II do CC/2002 (5 anos)
✓ Menciona pelo menos uma fonte legal
✓ Resposta em menos de 30 segundos
O caso expande e revela: a pergunta/passos a executar, os critérios de aprovação e o campo de anotações
✓
Execute os passos na plataforma Juris Code de verdade (juriscode.com.br) e depois volte para este formulário para marcar o resultado.

Passo 3

Marcando o Resultado

No canto direito de cada teste há três botões redondos. Clique no que corresponde ao resultado que você observou.
✓
APROVADO
A plataforma se comportou exatamente como descrito nos critérios de aprovação.
✗
REPROVADO
Algo não funcionou como esperado. Sempre descreva o que aconteceu nas anotações.
⊘
PULADO
Você não conseguiu ou não teve tempo de executar este teste agora.
i
O botão clicado fica destacado com a cor correspondente. Se clicar de novo no mesmo botão, a marcação é desfeita (fica pendente novamente).
i
Sua marcação é salva automaticamente no navegador — não existe botão "Salvar". Só não limpe o histórico/cache do navegador antes de exportar os resultados (Passo 5).
Exemplo: teste marcado como Aprovado

Anotações e Progresso

Abaixo dos critérios de cada teste há um campo de texto livre. Use-o sempre — mesmo quando o teste for aprovado.
Campo de anotações
Cole aqui a resposta da IA ou descreva o resultado...
✎
Quanto mais detalhes, mais fácil é para a equipe entender e corrigir. Prints de tela podem ser enviados depois, direto no grupo do WhatsApp, citando o código do teste (ex: "TC-007").
No topo da página, o Resumo de Progresso mostra em tempo real quantos testes você já avaliou — atualiza sozinho a cada clique.

Passo 5

Exportando e Enviando os Resultados

Quando terminar (ou sempre que quiser gerar um retrato do seu progresso), gere o relatório e envie para a equipe.
1
Preencha o campo "Seu nome" no topo da página (uma única vez) — ele identifica o relatório exportado.
2
Clique no botão "⬇ Exportar Resultados (.txt)", logo abaixo do Resumo de Progresso.
3
Um arquivo .txt é baixado automaticamente para a pasta de downloads do seu computador ou celular.
4
Envie esse arquivo para o grupo do WhatsApp Juris Code Beta ou por e-mail para contato@juriscode.com.br.
Botão de exportação
⬇ Exportar Resultados (.txt)
↺ Limpar tudo

Dicas Finais e Suporte

Resumo do fluxo completo
1. Abra o link do Protocolo → 2. Clique no título do teste para expandir → 3. Execute os passos na plataforma juriscode.com.br → 4. Marque ✓ / ✗ / ⊘ → 5. Escreva uma anotação → 6. Repita para os 40 testes (na sua ordem preferida) → 7. Exporte e envie o relatório.
💡
Não precisa fazer tudo de uma vez. Você tem 20 dias (21/07–10/08/2026) — divida os módulos ao longo dos dias.
💡
Os testes de IA (Módulos 02 e 03) consomem sua cota de 150 consultas compartilhadas do escritório — use com moderação, mas sem medo: dá para pedir mais se precisar.
💡
Encontrou algo que não está neste manual ou tem dúvida sobre um teste específico? Pergunte no grupo do WhatsApp — cite sempre o código do teste (ex: "TC-018").
Seu feedback é o produto Cada teste marcado e cada anotação escrita molda a versão final do Juris Code. Obrigado por fazer parte do beta.
