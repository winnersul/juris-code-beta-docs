💡 Dicas para Melhores Resultados
✅
Seja específico
"Calcule verbas rescisórias de CLT com 3 anos e salário R$4.500" funciona muito melhor que "calcule rescisão"
📎
Envie seus documentos
Quanto mais peças, contratos e jurisprudência você enviar, mais personalizado fica o resultado para o SEU escritório
🔄
Itere a conversa
Pode perguntar de volta: "Aprofunde o ponto 2" ou "Adapte para vara trabalhista". O contexto é mantido.
🗺️
Honorários: informe o estado
Para consultar honorários OAB, selecione sua UF. CE, SC e PR retornam apenas percentuais.
🤖
Escolha o agente certo
Para redigir peças → Redator. Para calcular → Perito. Para criar estratégia → Estrategista.
